# 📘 나의 평범한 일기장
2024년 글로벌미디어학부 졸업전시회 작품
> ‘나의 평범한 일기장’은 미니게임을 중심으로 진행되는 웹 게임입니다. <br>
> 전시의 주 관람객인 20대가 반가워할 2010년대 추억들을 그림 일기 컨셉으로 담아내었습니다. <br>
> 게임을 통해 특별하지 않아도 소중했던 어렸을 적 일상을 마주하고, 평범한 오늘도 추억으로 간직해 보면 어떨까요?
